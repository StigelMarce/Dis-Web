Botones personalizados

En este ejemplo se muestra cómo crear un botón personalizado con CSS.

Para crear un botón personalizado, se puede utilizar la clase .custom-button y aplicar los estilos deseados.

Al pasar el cursor sobre el botón, se puede cambiar el color de fondo utilizando la clase .custom-button:hover.
